Add Checkout Widget

Create an event widget for your website using one of your existing event templates.

Once you have customized the appearance of your widget, click Save Changes. This page will refresh to display a code that you can then copy and paste into your website to embed the widget.


Please note: Since we do not control your web properties (and we want to respect your security!), our team is not personally able to add the widget to your website. Please consult your webmaster if you experience any problems with the widget.
